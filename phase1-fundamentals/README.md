# Phase 1 — Go Fundamentals

Work through exercises in order. Each file has a `// TODO:` block with the task.

```bash
go run ./exercises/01_variables_structs
go test ./...
```

## Exercises

| # | Package | Topics |
|---|---------|--------|
| 01 | `exercises/01_variables_structs` | Variables, structs, interfaces |
| 02 | `exercises/02_pointers` | Pointers, pass by value vs reference |
| 03 | `exercises/03_methods` | Value vs pointer receivers, Stringer |
| 04 | `exercises/04_errors` | Error handling, wrapping, sentinel errors |
| 05 | `exercises/05_goroutines` | Goroutines, WaitGroup |
| 06 | `exercises/06_channels` | Channels, select, fan-in |
| 07 | `exercises/07_packages` | Packages, exported API |
| 08 | `exercises/08_context` | context.Context, cancellation |
| 09 | `exercises/09_testing` | Table-driven tests, benchmarks |

## Stdlib to read

`fmt`, `errors`, `sync`, `context`, `strings`, `testing`, `time`
